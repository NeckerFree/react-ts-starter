# SmartInventory UI (Practice)

## Goal
Practice modern React + TypeScript front-end patterns suitable for technical interviews:
- Vite + React 18 + TypeScript
- Tailwind CSS
- ESLint / Prettier
- Husky + lint-staged
- Strict TypeScript and path aliases
- Ready for adding React Query, MSW, tests, and CI

## Local setup
```bash
# install deps
npm install

# create husky hooks (runs on npm install because of "prepare" script)
npm run prepare

# dev server
npm run dev
